# Story Squares

**Story Squares** — веб-приложение для быстрого создания квадратных фото-историй/открыток из ваших снимков. Загружайте \~36 фотографий, раскладывайте по шаблону с разными слотами, добавляйте подписи и QR-ссылки, экспортируйте готовые карточки (JPEG/PNG) и общий PDF — всё локально, прямо в браузере.

> ⚠️ Проект в активной разработке (MVP). Этот README описывает цель и ближайший план; функциональность будет добавляться поэтапно.

---

## Зачем это нужно

* **Быстро** собрать серию квадратных карточек из набора фото.
* **Единый стиль**: фиксированные пресеты сетки/масок (слоты разного формата).
* **Подписи и QR**: пояснения к фото и ссылки (альбом/сайт/видео).
* **Экспорт офлайн**: готовые картинки + PDF без сервера и без сливов данных.

---

## Основные возможности (план MVP)

* Загрузка \~36 фотографий (drag-and-drop + выбор файлов).
* Авторазмещение по выбранному квадратному шаблону (слоты разных форм).
* Ручная подгонка: зум, позиция кадра, `cover/contain`, вкл/выкл масок.
* Подписи к фото и опциональный **QR-код** (генерация внутри приложения).
* Экспорт: пакет **JPEG/PNG** (на устройство/в ZIP) и общий **PDF**.
* Сохранение/загрузка проекта (локально, в браузере).
* Работа офлайн (PWA) и приватность: все данные остаются у пользователя.
